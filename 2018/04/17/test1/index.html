百里守约

  • 2018-04-17
  • 0
  • 0
  • 耶二鱼
  • CC BY-NC-ND 4.0

于时光深处，静看花开花谢，虽历尽沧桑，仍含笑一腔温暖如初。其实，不是不深情，是曾经情太深；不是不懂爱，是爱过知酒浓。生活的阡陌中，没有人改变得了纵横交错的曾经，只是，在渐行渐远的回望里，那些痛过的、都演绎成了坚强；那些不忍遗忘的、念念不忘的，都风干成了风景。
